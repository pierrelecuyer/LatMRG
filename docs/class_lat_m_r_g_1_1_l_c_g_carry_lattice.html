Detailed Description

template<typename Int, typename Dbl>
class LatMRG::LCGCarryLattice< Int, Dbl >

This class represents the lattice a of LCG with a carry defined by the recurrence

\[ x_n = a x_{n-1} +c \pmod m. \]

The building strategy for the lattice is technically the same, but MRGLattice does not feature the possibility to add one becausae it is seldom used for a MRG.

The class overrides the basis construction methods and implements a method to check if the parameter choice \((a,c,m)\) defines a full period LCG with carry.
