Detailed Description

template<typename Int, typename Dbl>
class LatMRG::MRGLatticeLac< Int, Dbl >

This class implements lattice bases built from multiple recursive linear congruential generators (see class MRGLattice) using lacunary indices.

Constructor with modulus of congruence \(m\), order of the recurrence \(k\), multipliers \(A\), maximal dimension maxDim, and lattice type latt.

Vector and matrix indices vary from 1 to maxDim. The length of the basis vectors is computed with norm. The bases are built using the lacunary indices lac.
