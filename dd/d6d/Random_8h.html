Detailed Description

This file offers functions to generate random numbers use when making random searches.

The base generator is the 64-bit LFSR258 from [rLEC99a], with period near \(2^{258}\), for 64-bit computers, and the 32-bit generator LFSR113 from [rLEC99a], with period length near \(2^{113}\) on 32-bits computers. Thus the random numbers are different on 32-bits and 64-bits machines.

NTL::ZZ LatticeTester::Random::randInt ( NTL::ZZ i,
NTL::ZZ j )

This is an overload of the previous method to create an NTL::ZZ integer in the range \([i,j]\).

NTL has a method to generate pseudo random numbers in a given range, but it is not deterministic. This method concatenates random bits until it has a big enough number and tries to assign the correct integer to it.
